# ⚔️ 烈火の剣 ダメージ計算ツール

ファイアーエムブレム烈火の剣のダメージ計算をブラウザ上で行えるツールです。

## 🎮 概要

自軍ユニットのステータスと武器を設定し、ゲーム内のボス敵との戦闘結果をシミュレーションできます。育成計画や攻略の参考にご活用ください。

## ✨ 機能

### ユニット設定
- **44名の自軍ユニット**から選択可能
- レベル1〜20の範囲で設定
- 各ステータス（HP/力/技/速さ/幸運/守備/魔防/体格）を個別に調整可能
- **期待値自動計算** - 成長率からレベルアップ後の期待ステータスを算出

### 武器
- **81種類の武器**を網羅
- 剣・槍・斧・弓・理魔法・光魔法・闇魔法に対応
- 英雄の武器（デュランダル、アルマーズ等）も収録

### クラスチェンジ
- CC前/CC後の状態を切り替え可能
- CCボーナス（ステータス上昇）を自動反映

### 戦闘計算
- **28体のボス敵**との戦闘結果を一括計算
- エリノー編/ヘクノー編でフィルタリング可能

## 📊 計算される項目

| 項目 | 説明 |
|------|------|
| 与ダメージ | 敵に与えるダメージ（×2は追撃あり） |
| 命中率 | 攻撃が当たる確率 |
| 必殺率 | クリティカル発生率 |
| 被ダメージ | 敵から受けるダメージ |
| 被命中率 | 敵の攻撃が当たる確率 |
| 被必殺率 | 敵のクリティカル発生率 |
| 評価 | 勝てそう/負けそう/決着つかず |

## 🔧 計算ロジック

### 武器相性
```
物理：剣 > 斧 > 槍 > 剣
魔法：理 > 光 > 闇 > 理

有利時：攻撃力+1、命中率+15
不利時：攻撃力-1、命中率-15
```

### 追撃判定
```
攻撃速度 = 速さ - (武器重量 - 体格)
※体格で相殺しきれない場合のみ減算

攻撃速度差が4以上で追撃発生
```

### 特効
- 弓 → 飛行系（ペガサスナイト、ドラゴンナイト等）：3倍
- マーニ・カティ、レイピア、ヴォルフバイル → 騎馬・重装：3倍
- アーマーキラー、つらぬきのやり、ハンマー → 重装：3倍
- デュランダル、ソール・カティ、アルマーズ → 竜：2倍

### クラス補正
- ソードマスター/狂戦士：必殺率+15

## 🚀 使い方

1. `index.html` をブラウザで開く
2. ユニットを選択
3. レベルを設定し「期待値を表示」で推奨ステータスを反映（任意で調整可）
4. 武器を選択
5. 「計算開始」をクリック
6. 難易度フィルターで結果を絞り込み（任意）

## 💻 動作環境

- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- サーバー不要（ローカルで動作）
- インターネット接続不要

## 📝 データについて

- ユニットデータ：初期ステータス、成長率、クラスチェンジ先
- 敵データ：エリノー編/ヘクノー編のボス敵
- 武器データ：威力、命中、必殺、重量、属性、特効
- クラスデータ：CCボーナス、ステータス上限

## ⚠️ 注意事項

- 本ツールはファンメイドの非公式ツールです
- ゲーム内の乱数や地形効果は考慮していません
- 一部の特殊武器（ルナの魔防無視など）は未対応です
